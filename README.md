# SWN Player Resources

Player-facing materials for an ongoing **Stars Without Number** campaign set in the Astyna Tau sector. Everything here is safe to read as a player. No GM secrets.

## The game

*Stars Without Number* is a far-future tabletop RPG of starship crews, lost worlds, and the long shadow of a fallen interstellar civilization. You don't need to own the rules to play, but if you want a copy:

- [Free rulebook (DriveThruRPG)](https://www.drivethrurpg.com/en/product/230009/stars-without-number-revised-edition-free-version) — the revised edition, free, complete enough to run and play characters with.

## Making a character

- [`character-creation.md`](character-creation.md) — the procedure and the numbers. Attributes, classes, skills, foci, hit points, saves, gear. Read this first.
- [`character-creation-descriptions.md`](character-creation-descriptions.md) — companion reference. What each class, skill, and focus actually *means*, so you can choose informedly.
- [Online character generator](https://swnfreebooter.net/characterDesignerDemo) — a third-party web tool that walks you through character creation step by step. Good if you'd rather click through it than do it on paper. The output is straightforward to transcribe.

## The setting

- [`setting-overview.md`](setting-overview.md) — the default *Stars Without Number* setting. The Terran Mandate, the Scream, the Silence, the present day. Background for any SWN campaign.
- [`astyna-tau-sector/herdisan-reach-players-guide.md`](astyna-tau-sector/herdisan-reach-players-guide.md) — a traveler's guide to the Herdisan Reach, the cluster of six inhabited systems where the campaign takes place. Written as in-world common knowledge: what spacers, merchants, and anyone who's spent time in the region knows.

## The campaign

- [`starting-ship-choices.md`](starting-ship-choices.md) — three options for how your crew came into possession of a ship. Pick one before session one. Each has a downside; none of them are clean.
